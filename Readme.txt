Программа Magic2 для обработки измерений с ИС: "Гамма-156", "Schuster TSM 664", "GAMMA TSSemi 2000-400".

История версий:

1.2.0.0: 
- Первая версия.

1.3.0.0: 
- Исправлена ошибка Количество годных (иногда не правильно считала).
- Добавлен лист "Нормы и режимы".
- Добавлен режим карты годности по параметрам.
- Добавлен "Schuster TSM 664".

1.3.1.0: 
- Для"Гамма-156": исправлена ошибка с отображением карт годности.

1.3.2.0:

- Для "Schuster TSM 664": обход приведён к правильному. 
- Для "Schuster TSM 664": добавлена возможность добавления карты обхода отдельным файлом .map (поменять расширение уже измеренного файла).

1.3.2.2:
- Для "Гамма-156": добавлена возможность загружать карту обхода из любого измеренного файла .sts.

- Для "Гамма-156": если измеренный параметр = 0, значит он не измерялся.
- Для всех: при выборе карты годности по параметрам также отображается общая карта годности.
- Для всех: добавлен лист "Гистограммы" с гистограммами по всем измерениям и параметрам

1.4.1.1:
